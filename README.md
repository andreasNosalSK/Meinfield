<p>
    Hra Minesweeper predstavuje klasickú logickú úlohu, v ktorej hráč odkrýva jednotlivé políčka herného poľa s cieľom označiť všetky míny bez toho, aby na ne klikol. Dokumentácia popisuje architektúru celého programu prostredníctvom UML diagramu, detailný opis jednotlivých tried, ich zodpovednosti a vzájomné vzťahy.
